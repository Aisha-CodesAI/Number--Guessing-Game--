# 🎮 Number Nexus – The Number Guessing Game

A fun and interactive Python-based Number Guessing Game developed to improve logical thinking and programming skills. The game generates a random number, and the player must guess the correct number within limited attempts using smart hints provided by the system.

---

## ✨ Features

- Multiple Difficulty Levels  
- Smart Hint System  
- Score Calculation  
- Timer Functionality  
- Colored Terminal Output  
- Input Validation  
- Scoreboard System  
- User-Friendly CLI Interface  

---

## 🛠 Technologies Used

- Python 3.x  
- Random Module  
- Time Module  
- Colorama Library  

---

## 🚀 How to Run

### 1. Clone the Repository

```bash
git clone <your-github-repo-link>
```

### 2. Open Project Folder

```bash
cd Number-Nexus
```

### 3. Install Required Library

```bash
pip install colorama
```

### 4. Run the Program

```bash
python number_nexus.py
```
